## 🛠️ Installation Steps

Star and Fork the Repo 🌟 and this will keep us motivated.

1. Clone the repository

```bash
git clone https://github.com/adityapal87/Typing-test
```

2. Change the working directory

```bash
cd Typing-test
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run start
```
